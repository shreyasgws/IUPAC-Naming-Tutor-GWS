1. UNDERSTANDING COMPRESSED FORMULA OF ORGANIC COMPOUNDS

🔹 What is a Compressed Formula?

👉 A compressed formula is a shorthand way to write organic compound structures. It shows how atoms are connected without drawing every bond.

In organic chemistry, we use different types of compressed formulas to represent molecules.

🔹 1. Molecular Formula

👉 Shows the actual number of each type of atom in the molecule.

Formula: CxHyOz...

Examples:

  • C2H6 → Ethane (2 carbons, 6 hydrogens)
  • C3H8 → Propane (3 carbons, 8 hydrogens)
  • C4H10 → Butane (4 carbons, 10 hydrogens)

⚠️ Limitation: Molecular formula does NOT show the structure or connectivity of atoms. Different compounds can have the same molecular formula!

🔹 2. Structural Formula (Full Structural)

👉 Shows all bonds between atoms clearly.

Examples:

  • Methane: CH4
    CH4
  • Ethane: CH3–CH3
    CH3–CH3
  • Propane: CH3–CH2–CH3
    CH3–CH2–CH3

🔹 3. Condensed Formula (Most Important)

👉 Groups atoms to show connectivity without drawing all bonds. The most commonly used format in IUPAC questions.

📌 Rule 1: Parentheses for Branches

👉 Atoms attached as branches are enclosed in parentheses.

Examples:

  • CH3–CH(CH3)–CH3 → Isobutane (2-Methylpropane)
    • Interpretation: Carbon 1 has 3H, Carbon 2 has 1H + branch (CH3), Carbon 3 has 3H
  • CH3–C(CH3)2–CH3 → Neopentane (2,2-Dimethylpropane)
    • Interpretation: Central carbon has 2 methyl branches

📌 Rule 2: Repeating Groups

👉 Subscripts outside parentheses show repeated groups.

Examples:

  • CH3–(CH2)3–CH3 → Hexane (n = 6)
    • CH3–(CH2)3–CH3 = CH3–CH2–CH2–CH2–CH3
  • CH3–(CH2)4–CH3 → Heptane (n = 7)

📌 Rule 3: Functional Groups in Condensed Form

Examples:

GroupCondensed FormStructure

📌 Rule 4: Multiple Bonds

Examples:

  • CH2=CH–CH3 → Propene (double bond between C1-C2)
  • CH3–C≡C–CH3 → 2-Butyne (triple bond)

🔹 How to Read Condensed Formulas – Step by Step

Step 1: Identify the Main Chain

👉 Start from left, follow the main chain (atoms not in parentheses).

Step 2: Count Branched Groups

👉 Each set of parentheses represents a substituent branch.

Step 3: Calculate Hydrogen Count

👉 Each carbon needs 4 bonds. Count existing bonds from the condensed form.

Example Analysis:

CH3–CH(CH3)–CH2–CH3

  • Main chain: CH3–CH–CH2–CH3 = 4 carbons = Butane
  • Branch: (CH3) on 2nd carbon = Methyl group
  • Result: 2-Methylbutane

Practice Analysis:

CH3–C(CH3)2–CH2–CH(CH3)–CH3

  • Main chain: 5 carbons = Pentane
  • Branches:
    • 2nd carbon: Two methyl groups (CH3)2
    • 4th carbon: One methyl group
  • Result: 2,2,4-Trimethylpentane

🔹 Special Cases in Compressed Formulas

Case 1: Functional Groups with Subscripts

  • HO–CH2–CH2–OH → Ethane-1,2-diol (Ethylene glycol)
  • HOOC–CH2–COOH → Propane-1,3-dioic acid

Case 2: Rings and Cyclic Compounds

  • (CH2)6 with ring closure → Cyclohexane

Case 3: Aromatic Compounds

  • C6H5–CH3 → Toluene (Methylbenzene)
  • C6H5–OH → Phenol

2. BOND LINE STRUCTURE (SKELETAL FORMULA) OF ORGANIC COMPOUNDS

🔹 What is Bond Line Structure?

👉 Bond Line Structure (also called Skeletal Formula) is the most simplified way to represent organic molecules. It's what chemists use in practice because it's fast and clear.

⚡ Key Principle: In bond line structure, carbon atoms are NOT shown explicitly. Only the bonds (lines) are drawn.

🔹 Rule 1: Carbon Atoms Are Implied

👉 Each corner and endpoint in a bond line structure represents a carbon atom.

Examples:

  • A straight line with 2 endpoints: —— = 2 carbons = Ethane
  • A zigzag line: —\/— (bent line) = 3 carbons = Propane
  • A zigzag line with 4 segments: = 4 carbons = Butane

🔹 Rule 2: Hydrogen Atoms Are Implied

👉 Hydrogen atoms are NOT shown. Each carbon automatically has enough hydrogens to make 4 bonds total.

Example Analysis:

  • Endpoint carbon: Has 3 implied H atoms (1 bond shown + 3 H = 4 bonds)
  • Middle carbon in chain: Has 2 implied H atoms (2 bonds shown + 2 H = 4 bonds)
  • Corner carbon: Has 2 implied H atoms (2 bonds shown + 2 H = 4 bonds)

💡 To find H count: 4 bonds per carbon - bonds shown = H atoms

🔹 Rule 3: Multiple Lines for Multiple Bonds

Examples:

Bond TypeRepresentationExample
Single BondOne line (—)Ethane
Double BondTwo parallel lines (=)Ethene
Triple BondThree parallel lines (≡)Ethyne

Practical Examples:

  • Propene: —=— →
  • 2-Butene: —=— with chain →
  • But-1-en-3-yne: —=—≡— →

🔹 Rule 4: Non-Carbon Atoms Must Be Shown

👉 All atoms OTHER than carbon and hydrogen MUST be explicitly written.

Examples:

AtomSymbolExample

NitrogenNAmine: —NH2
SulfurSThiol: —SH

🔹 Rule 5: Functional Groups with Multiple Atoms

Examples:

  • Aldehyde (–CHO): Write as —CHO
  • Ketone (>C=O): Write as —C— with double bond O
  • Carboxylic Acid (–COOH): Write as —COOH
  • Ester (–COOR): Write as —COOR

🔹 How to Count Carbons in Bond Line Structure

Method 1: Count Endpoints and Corners

Example:

Each endpoint and corner represents 1 carbon atom.

  • —— (2 endpoints) = 2 carbons
  • ——|—— (3 corners/endpoints) = 3 carbons
  • | | (2 corners + 2 endpoints) = 4 carbons

Method 2: Count Line Segments

👉 Number of carbons = Number of line segments

Example:

A zigzag line with 5 segments = 5 carbons (Pentane)

Method 3: Double/Triple Bonds Don't Add Carbons

👉 Multiple lines between same atoms don't create new carbons.

Example:

—— with double bond = 2 carbons with double bond = Ethene

🔹 Branches in Bond Line Structure

👉 Branches are drawn as lines coming off the main chain at angles.

Example: 2-Methylbutane

Structure: A 4-carbon chain with a methyl branch at carbon 2

Notice: The branch (methyl) comes off as a short line from the 2nd carbon.

Example: 2,2-Dimethylpropane (Neopentane)

Structure: Central carbon with 4 methyl groups

Example: 3-Ethyl-2-methylpentane

Structure: 5-carbon chain with ethyl and methyl branches

🔹 Cyclic Compounds in Bond Line

👉 Rings are drawn as polygons (triangles, squares, pentagons, hexagons, etc.)

Examples:

👉 Benzene is drawn as a hexagon with a circle inside (or alternating double bonds).

Benzene representations:

3. IUPAC NAMING – BASIC RULES (WITH EXAMPLES)

🔹 3. Select the Longest Carbon Chain (Parent Chain)

👉 Choose the longest continuous chain of carbon atoms.

Examples:

  • CH3–CH2–CH3 → Propane (3 carbons)
  • CH3–CH(CH3)–CH2–CH3 → longest chain = 4 → Butane base

👉 Special case: If multiple chains have same length → choose the one with more substituents.

🔹 4. Choose Chain with Functional Group / Multiple Bond

👉 Parent chain must include:

  • Functional group
  • Double/triple bond

Examples:

  • CH3–CH2–CH2–OH → chain must include –OH → Propanol
  • CH2=CH–CH2–CH3 → include double bond → Butene

🔹 5. Number the Carbon Chain (Lowest Locant Rule)

👉 Number from the end that gives lowest number to: Functional group, Double/triple bond, Substituents.

Examples:

  • (a) Functional group priority: CH3–CH2–CH2–OH → Number from –OH side → Propan-1-ol (not propan-3-ol)
  • (b) Double bond priority: CH2=CH–CH2–CH3 → But-1-ene (not but-3-ene)
  • (c) Substituent case: CH3–CH(CH3)–CH2–CH3 → 2-Methylbutane

🔹 6. Identify and Name Substituents

👉 Groups attached to main chain = substituents

GroupName
–CH3methyl
–C2H5ethyl
–Clchloro

Examples:

  • CH3–CH(CH3)–CH3 → 2-Methylpropane
  • CH3–CH(Cl)–CH3 → 2-Chloropropane

🔹 7. Assign Position Numbers (Locants)

👉 Write number of carbon where substituent is attached

Examples:

  • CH3–CH(CH3)–CH3 → 2-methylpropane
  • CH3–CH2–CH(Cl)–CH3 → 2-chlorobutane

🔹 8. Multiple Same Substituents → Use Prefixes

👉 di-, tri-, tetra-

Examples:

  • CH3–C(CH3)2–CH3 → 2,2-Dimethylpropane
  • CH3–CH(CH3)–CH(CH3)–CH3 → 2,3-Dimethylbutane

🔹 9. Arrange Substituents Alphabetically

👉 Ignore di-, tri- while ordering

Example:

  • CH3–CH(CH3)–CH(C2H5)–CH3 → 3-Ethyl-2-methylbutane ✔ (not methyl before ethyl ❌)

🔹 10. Identify Type of Bond (Primary Suffix)

BondSuffix
Single-ane
Double-ene
Triple-yne

Examples:

  • CH3–CH3 → Ethane
  • CH2=CH2 → Ethene
  • CH≡CH → Ethyne

🔹 11. Write Position of Double/Triple Bond

👉 Always mention position

Examples:

  • CH2=CH–CH3 → Prop-1-ene
  • CH3–CH=CH–CH3 → But-2-ene

🔹 12. Add Functional Group (Secondary Suffix)

GroupSuffix
–OH-ol
–CHO-al
>C=O-one
–COOH-oic acid

Examples:

  • CH3–CH2–OH → Ethanol
  • CH3–CHO → Ethanal
  • CH3–CO–CH3 → Propanone
  • CH3-CO-C2H5 → Butan-2-one ⭐

🔹 13. Functional Group Priority (Very Important)

👉 Highest priority gets: Lowest number, Suffix

Order: COOH > CHO > CO > OH > Double bond > Alkyl

Examples:

  • (a) HO–CH2–CHO → Aldehyde priority → 2-Hydroxyethanal
  • (b) CH3–CO–CH2–OH → Ketone priority → 3-Hydroxybutan-2-one

🔹 14. Multiple Functional Groups

👉 Highest → suffix, Others → prefix

Example:

  • HO–CH2–CH2–COOH → 3-Hydroxypropanoic acid

🔹 15. Naming Cyclic Compounds

👉 Add cyclo-

Examples:

  • Cyclopropane
  • Cyclohexane

🔹 16. Naming Aromatic Compounds (Basic)

Examples:

  • C6H6 → Benzene
  • C6H5–CH3 → Methylbenzene
  • C6H5–OH → Phenol

🔹 17. Special Case – Tie Breaking Rule

👉 If two numbering options give same numbers, choose one with: More substituents, Lower numbers at first difference.

Example:

  • CH3–CH(CH3)–CH(CH3)–CH3 → 2,3-Dimethylbutane

⚡ Final Example (Mixed Concept - Section 3)

Compound: CH3–CH(CH3)–CH2–CH2–OH

  1. Longest chain = 4 → butane
  2. Functional group = OH
  3. Number from OH side
  4. Substituent = methyl

👉 Final name: 3-Methylbutan-1-ol

4. SPECIAL CASES IN IUPAC NAMING

🔹 1. When Two Longest Chains Are Possible

👉 If more than one chain has the same length: Choose the chain with maximum substituents

Example: CH3–CH(CH3)–CH(CH3)–CH3

Two possible 4-carbon chains exist. ✔ Choose the one with more branches.

👉 2,3-Dimethylbutane

🔹 2. When Numbering from Both Ends Gives Same Set

👉 Use first point of difference rule

Example: CH3–CH(CH3)–CH2–CH(CH3)–CH3

Left → 2,4 | Right → 2,4

Tie! Now check next difference: ✔ Choose numbering giving lower number to first substituent.

👉 Correct: 2,4-Dimethylpentane

🔹 3. Functional Group vs Multiple Bond Conflict

👉 Functional group gets priority over double/triple bond

Example: CH2=CH–CH2–OH

Two options: Double bond at 1, OH at 1. ✔ OH gets priority.

👉 Name: Prop-2-en-1-ol

🔹 4. Multiple Bonds (Double vs Triple)

👉 Double bond gets preference in numbering

Example: CH2=CH–C≡CH

Numbering: ✔ Give double bond lower number.

👉 Name: But-1-en-3-yne

🔹 MULTIPLE DOUBLE / TRIPLE BONDS – RULES

🔹 i. Choose the Longest Chain Including Maximum Multiple Bonds

👉 Parent chain must contain: Maximum number of double/triple bonds

Example: CH2=CH–CH=CH2

✔ Chain includes both double bonds → Base name: buta-…

🔹 ii. Use Special Suffixes

👉 For multiple bonds:

BondsSuffix
2 double bonds-diene
3 double bonds-triene
2 triple bonds-diyne

Examples:

  • CH2=CH–CH=CH2 → Buta-1,3-diene ⭐
  • CH≡C–C≡CH → Buta-1,3-diyne

🔹 5. When Functional Group Is Not in Main Chain

👉 You MUST include it in parent chain

Example: CH3–CH(CH3)–CH2–CHO

Even if a longer chain exists, include –CHO.

👉 Name: 3-Methylbutanal

🔹 6. Multiple Functional Groups (Different Types)

👉 Highest priority = suffix | Others = prefix

Example: HO–CH2–CH2–CHO

✔ Aldehyde > alcohol.

👉 Name: 3-Hydroxypropanal (Note: original says 2-Hydroxyethanal, updated to match formula)

🔹 7. Same Functional Group Repeated

👉 Use di-, tri-, etc.

Example: HO–CH2–CH2–OH

👉 Name: Ethane-1,2-diol

🔹 8. Branched Substituents (Complex Substituents)

👉 When substituent itself is branched: Number substituent separately. Use brackets.

Example: CH3–CH(CH2CH3)–CH2–CH3

👉 Name: 3-Methylpentane (Note original: 2-Ethylbutane)

More complex: CH3–CH(CH(CH3)2)–CH3

👉 Name: 2-(Propan-2-yl)propane (Note: commonly just 2,3-Dimethylbutane)

🔹 9. Alphabetical Order with Prefixes

👉 Ignore: di-, tri-, tetra-
👉 Consider: iso (counts), neo (counts)

Example: CH3–CH(CH3)–CH(C2H5)–CH3

👉 Name: 3-Ethyl-2-methylbutane

🔹 10. Numbering in Cyclic Compounds

👉 Start numbering from: Functional group, then give lowest numbers to substituents.

Examples:

  • Cyclohexane with one methyl group → Methylcyclohexane
  • Two substituents: 👉 Use lowest set → 1,2-dimethylcyclohexane ✔ (not 1,5 ❌)

🔹 11. Aromatic Compounds (Special Positions)

👉 Use: ortho (1,2), meta (1,3), para (1,4)

Example: C6H4(CH3)2

  • 1,2 → o-xylene
  • 1,3 → m-xylene
  • 1,4 → p-xylene

🔹 12. When No Functional Group (Only Alkyl + Multiple Bonds)

👉 Double/triple bond gets priority in numbering.

Example: CH3–CH=CH–CH3

👉 But-2-ene

🔹 13. When Substituent and Functional Group Compete

👉 Functional group always wins.

Example: CH3–CH(Cl)–CH2–OH

✔ OH gets lowest number.

👉 Name: 2-Chloropropan-1-ol

🔹 14. Longest Chain vs Maximum Functional Groups

👉 Choose chain with maximum functional groups, even if slightly shorter.

Example: A chain of 5 carbons vs chain of 4 carbons with –OH.

✔ Choose 4-carbon chain with OH.

🔹 15. Writing Format (Common Mistake Case)

👉 Correct format: Numbers → commas, Number + word → hyphen.

Example:

  • ✔ 2,3-dimethylbutane
  • ❌ 2 3 dimethyl butane

⚡ Most Important Special Cases for Exams (Section 4)

Focus on these:

  • Tie-breaking (same numbering)
  • Functional group priority
  • Double vs triple bond
  • Multiple substituents
  • Branched substituents
  • Aromatic naming (o, m, p)

5. PRIORITY ORDER IN IUPAC NAMING

🔹 What is Priority Order?

👉 When a compound has multiple functional groups, priority order decides: Which group gets the suffix (main group), Which groups become prefixes, Where to start numbering.

🔝 STANDARD PRIORITY ORDER (You must memorize)

–COOH (Carboxylic acid)
⬇
–CHO (Aldehyde)
⬇
>C=O (Ketone)
⬇
–OH (Alcohol)
⬇
–NH2 (Amine - sometimes included)
⬇
C=C / C≡C (Double/triple bond)
⬇
–R, –X, –NO2 (Alkyl, halogen, nitro)

🔹 KEY RULES

✅ Rule 1: Highest Priority → Suffix

👉 The group with highest priority is written as suffix.

Example: CH3–CH2–COOH + –OH

✔ COOH has higher priority.

👉 Name: Hydroxypropanoic acid (not propanol ❌)

✅ Rule 2: Lower Priority → Prefix

👉 All other groups become prefixes.

Example: HO–CH2–CH2–CHO

✔ Aldehyde > alcohol.

👉 Name: 3-Hydroxypropanal (Text states 2-Hydroxyethanal contextually)

✅ Rule 3: Numbering Starts from Highest Priority

👉 Give lowest number to main functional group.

Example: CH3–CH2–CH2–OH

✔ Number from OH side.

👉 Propan-1-ol

✅ Rule 4: Priority Over Multiple Bonds

👉 Functional group > double/triple bond.

Example: CH2=CH–CH2–OH

Two possibilities: Double bond at 1, OH at 1. ✔ OH wins.

👉 Prop-2-en-1-ol

✅ Rule 5: Between Double and Triple Bond

👉 Double bond gets priority in numbering.

Example: CH2=CH–C≡CH

👉 But-1-en-3-yne

🔹 DIFFERENT SCENARIOS (VERY IMPORTANT)

  • 🟡 Scenario 1: Acid + Alcohol HO–CH2–CH2–COOH
    ✔ COOH highest → 3-Hydroxypropanoic acid
  • 🟡 Scenario 2: Aldehyde + Ketone CH3–CO–CH2–CHO
    ✔ Aldehyde > ketone → 3-Oxobutanal
  • 🟡 Scenario 3: Alcohol + Ketone CH3–CO–CH2–OH
    ✔ Ketone > alcohol → 1-Hydroxypropan-2-one (Or 3-Hydroxypropan-2-one depending on context)
  • 🟡 Scenario 4: Only Multiple Bonds CH2=CH–CH=CH2
    → Buta-1,3-diene
  • 🟡 Scenario 5: Halogen + Alcohol CH3–CH(Cl)–CH2–OH
    ✔ Alcohol > halogen → 2-Chloropropan-1-ol
  • 🟡 Scenario 6: Amine + Alcohol HO–CH2–CH2–NH2
    ✔ Alcohol > amine → 2-Aminoethan-1-ol (2-Aminoethanol)
  • 🟡 Scenario 7: Carboxylic Acid + Everything CH3–CH(OH)–COOH
    👉 Acid ALWAYS dominates → 2-Hydroxypropanoic acid

⚡ FINAL MASTER EXAMPLE (Section 5 - Priority)

Compound: HO–CH2–CH(CH3)–CHO

  1. Functional groups: OH & CHO
  2. Priority: CHO > OH
  3. Parent chain: 3 carbons
  4. Number from CHO

👉 Final name: 3-Hydroxy-2-methylpropanal

6. PREFIX & SUFFIX OF FUNCTIONAL GROUPS

🔹 1. Basic Idea

👉 In IUPAC naming:

  • Suffix → used for the main (highest priority) functional group
  • Prefix → used for other (lower priority) groups

Example: HO–CH2–CH2–CHO

–CHO (aldehyde) → suffix (-al)
–OH (alcohol) → prefix (hydroxy-)

👉 Name: 3-Hydroxypropanal (Note: The prompt example had an error, corrected based on structure: 3-Hydroxypropanal).

🔹 2. Common Functional Groups (Prefix & Suffix Table)

🔹 3. How to Use Suffix

👉 Use suffix ONLY for:

  • Highest priority group
  • At end of name

Examples:

  • (a) Alcohol as main group: CH3–CH2–OH → Ethanol
  • (b) Ketone as main group: CH3–CO–CH3 → Propanone
  • (c) Acid as main group: CH3–COOH → Ethanoic acid

🔹 4. How to Use Prefix

👉 Use prefix when:

  • Functional group is NOT highest priority

Examples:

  • (a) Alcohol as prefix: HO–CH2–CHO → 2-Hydroxyethanal
  • (b) Ketone as prefix: CH3–CO–CH2–CHO → 3-Oxobutanal
  • (c) Amine as prefix: CH3–CH2–CH2–OH + NH2 (Correct formula: NH2–CH2–CH(OH)–CH3) → 1-Aminopropan-2-ol

🔹 5. Multiple Prefixes Together

👉 Write all prefixes in alphabetical order

Example: CH3–CH(Cl)–CH(OH)–CH3

👉 Name: 3-Chloro-2-hydroxybutane

🔹 6. Multiple Same Functional Groups

👉 Use di-, tri- etc.

Examples:

  • (a) Two alcohol groups: HO–CH2–CH2–OH → Ethane-1,2-diol
  • (b) Two aldehydes: OHC–CH2–CHO → Propane-1,3-dial

🔹 7. Special Case: Acid Group

👉 Acid has special naming:

  • No “e” from alkane
  • Add -oic acid

Example: CH3–CH2–COOH

👉 Name: Propanoic acid

🔹 8. Special Case: Aldehyde & Ketone Prefix Names

👉 When used as prefix:

  • Aldehyde → formyl-
  • Ketone → oxo-

Examples:

  • (a) CH3–CO–CH2–CHO → 3-Oxobutanal
  • (b) CHO–CH2–COOH → 3-Oxopropanoic acid (Note: IUPAC allows formylethanoic acid but oxopropanoic is preferred for acyclic).

🔹 9. Ether Naming (Prefix Only)

👉 Ether always uses prefix form

Examples:

  • CH3–O–CH3 → Methoxymethane
  • CH3–O–C2H5 → Methoxyethane

🔹 10. Halogens (Prefix Only)

👉 Always prefixes

Examples:

  • CH3–Cl → Chloromethane
  • CH3–CH(Br)–CH3 → 2-Bromopropane

⚡ 11. Mixed Functional Group Example (Full Application)

Compound: HO–CH2–CH(Cl)–COOH

  1. Highest priority = COOH → suffix
  2. Others → prefixes (hydroxy, chloro)
  3. Alphabetical order

👉 Final Name: 3-Chloro-2-hydroxypropanoic acid

⚡ Quick Memory Trick
👉 Suffix = King 👑 (highest priority)
👉 Prefix = Helpers

7. NAMING OF CYCLIC COMPOUNDS

🔹 1. Basic Rule (Cycloalkanes)

👉 Add prefix “cyclo-” to the alkane name

Examples:

  • 3-member ring → Cyclopropane
  • 4-member ring → Cyclobutane
  • 6-member ring → Cyclohexane

🔹 2. Single Substituent on Ring

👉 No numbering needed (position = 1 by default)

Example: A methyl group on cyclohexane

👉 Methylcyclohexane

🔹 3. Two Substituents on Ring

👉 Number to give lowest set of locants

Examples:

  • Adjacent → 1,2-dimethylcyclohexane
  • One carbon gap → 1,3-dimethylcyclohexane
  • Opposite → 1,4-dimethylcyclohexane

🔹 4. Different Substituents

👉 Use: Lowest numbers, Alphabetical order.

Example: Cyclohexane with ethyl and methyl

👉 1-Ethyl-3-methylcyclohexane

🔹 5. Ring vs Chain (Important Special Case)

👉 Choose as parent: Ring → if it has more carbons than chain

Examples:

  • (a) Small chain + big ring: 👉 Methylcyclohexane
  • (b) Large chain + small ring: 👉 Ring becomes substituent → 1-Cyclopropylpropane

🔹 6. Cyclic Compounds with Double Bond

👉 Double bond gets position 1

Examples:

  • One double bond → Cyclohexene
  • Two double bonds → Cyclohexa-1,3-diene

🔹 7. Functional Group in Ring

👉 Functional group gets: Lowest number, Suffix.

Example: Cyclohexane with –OH

👉 Cyclohexanol

8. BENZENE & ITS DERIVATIVES

🔹 1. Basic Compound

👉 C6H6 = Benzene

🔹 2. Monosubstituted Benzene

👉 Only one substituent → no numbering needed

Examples:

  • CH3–C6H5 → Methylbenzene (toluene) ⭐
  • Cl–C6H5 → Chlorobenzene
  • OH–C6H5 → Phenol ⭐
  • NH2–C6H5 → Benzenamine (Aniline) ⭐

🔹 3. Disubstituted Benzene (Very Important)

👉 Use positions:

PositionsName
1,2ortho (o-)
1,3meta (m-)
1,4para (p-)

Examples:

  • 1,2-dimethylbenzene → o-xylene ⭐
  • 1,3-dimethylbenzene → m-xylene ⭐
  • 1,4-dimethylbenzene → p-xylene ⭐

🔹 4. Numbering in Disubstituted Benzene

👉 Choose: Lowest numbers, Alphabetical order.

Example: Cl and NO2 on benzene

👉 1-Chloro-3-nitrobenzene

🔹 5. When One Group Has Special Name

👉 Some groups define parent name:

  • –OH → Phenol
  • –NH2 → Aniline
  • –CH3 → Toluene
  • –COOH → Benzoic acid

Example: OH + CH3

👉 Parent = phenol → 2-Methylphenol (o-cresol)

🔹 6. More Than Two Substituents

👉 Use numbering (not o/m/p)

Example: Three methyl groups

👉 1,3,5-Trimethylbenzene (Mesitylene)

🔹 7. Benzene as Substituent

👉 When benzene is attached to chain: Use phenyl-

Example: C6H5–CH2–CH3

👉 Phenylethane

🔹 8. Functional Group Priority in Benzene

👉 Same priority rules apply

Example: –COOH + –OH

👉 COOH wins → Hydroxybenzoic acid

⚡ MIXED EXAMPLES (IMPORTANT)

  • Example 1: Cyclohexane + OH + CH3 → OH priority → suffix → 2-Methylcyclohexanol
  • Example 2: Benzene + NO2 + Cl → Alphabetical order → 1-Chloro-3-nitrobenzene
  • Example 3: Cyclohexene + methyl → Double bond = 1 → 3-Methylcyclohexene

9. WHAT IS XYLENE?

🔹 Overview

👉 Xylene = Dimethylbenzene

It means:

  • A benzene ring (C6H6)
  • With two methyl (–CH3) groups attached

👉 General formula: C6H4(CH3)2

🔹 Types of Xylene (Very Important)

Depending on positions of the two –CH3 groups:

1. Ortho-xylene (o-xylene)

👉 Positions: 1,2-

Methyl groups are adjacent

👉 IUPAC name: 1,2-Dimethylbenzene

2. Meta-xylene (m-xylene)

👉 Positions: 1,3-

One carbon gap between groups

👉 IUPAC name: 1,3-Dimethylbenzene

3. Para-xylene (p-xylene)

👉 Positions: 1,4-

Opposite sides of ring

👉 IUPAC name: 1,4-Dimethylbenzene

🔹 Summary Table

🔹 Key Points for Exams:

  • “Xylene” always means two methyl groups on benzene
  • o-, m-, p- indicate relative positions
  • For 3 or more substituents → don’t use o/m/p, use numbers

10. ADVANCED CONCEPTS & RULES

1. LOWEST SET OF LOCANTS RULE (VERY IMPORTANT)

👉 Rule: When multiple numbering options exist, compare the entire set of locants, not just the first number.

👉 Choose the set with: Lowest number at first point of difference

Example: CH3–CH(CH3)–CH(CH3)–CH(CH3)–CH3

Possible numbering: Left → 2,3,4 | Right → 2,3,4 (same)

👉 No difference → both valid

Trickier Example: CH3–CH(CH3)–CH2–CH(CH3)–CH2–CH3

Options: Left → 2,4 | Right → 3,5

✔ Compare: 2 vs 3 → 2 is smaller

👉 Final Name: 2,4-Dimethylhexane

🧪 2. COMPLEX (BRANCHED) SUBSTITUENTS

👉 Rule: If substituent itself is branched: Treat it as a mini chain, Number it separately, Write in brackets

Example 1: CH3–CH(CH(CH3)2)–CH3

Substituent = isopropyl

👉 IUPAC: 2-(Propan-2-yl)propane (Usually just 2,3-Dimethylbutane)

Another Example: CH3–CH(CH2CH3)–CH2–CH3

👉 Name: 3-Methylpentane (Note text read 2-Ethylbutane)

🧪 3. ISO / NEO / SEC / TERT (COMMON → IUPAC)

👉 Rule: These are common names, convert to IUPAC in exams

CommonIUPAC
IsopropylPropan-2-yl
Isobutane2-Methylpropane
Neopentane2,2-Dimethylpropane
tert-butyl1,1-Dimethylethyl

Example: (CH3)3C– → tert-butyl

👉 IUPAC: 1,1-dimethylethyl

🧪 4. POLYFUNCTIONAL COMPOUNDS (3+ GROUPS)

👉 Rule: Identify highest priority → suffix, Others → prefixes, Number accordingly

Example 1: HO–CH2–CH(NH2)–COOH

Priority: COOH > OH > NH2

👉 Name: 2-Amino-3-hydroxypropanoic acid

Another Example: CH3–CO–CH(OH)–CH2–COOH

👉 Name: 3-Hydroxy-4-oxopentanoic acid (Text read 4-Hydroxy-3-oxopentanoic acid, updated to match carbon chain priority)

🧪 5. DETAILED NAMING OF ETHERS

👉 Rule: Use alkoxy + alkane (Smaller group → alkoxy, Larger chain → parent)

Examples:

  • (a) CH3–O–CH2–CH3 → Methoxyethane
  • (b) C2H5–O–C3H7 → Ethoxypropane

🔹 Special Case: Symmetrical ether

CH3–O–CH3 → Methoxymethane

🧪 6. CARBOXYLIC ACID DERIVATIVES (BASIC IDEA)

(Only basic awareness needed)

🔹 Ester (–COOR)

Example: CH3–COO–CH3

👉 Name: Methyl ethanoate

🔹 Naming pattern: Alkyl (from alcohol) + acid name

🧪 7. BENZENE WITH 3 OR MORE SUBSTITUENTS

👉 Rule: Do NOT use o/m/p | Use numbers only

  • Three methyl groups on benzene: 👉 1,3,5-trimethylbenzene
  • Another Example: Cl, NO2, CH3 → 1-Chloro-2-methyl-4-nitrobenzene (Dependent on positions)

🧪 8. PHENYL vs BENZYL (VERY IMPORTANT)

GroupName
C6H5–phenyl
C6H5–CH2–benzyl

Examples:

  • (a) C6H5–CH3 → Methylbenzene
  • (b) C6H5–CH2–Cl → Benzyl chloride

🔹 Trick: 👉 If CH2 present before ring → benzyl

🧪 9. ALDEHYDE ON BENZENE (SPECIAL)

👉 Rule: Use benzaldehyde (NCERT standard)

Examples:

  • C6H5–CHO → Benzaldehyde
  • With substituent → 2-Hydroxybenzaldehyde

🧪 10. FINAL WRITING FORMAT (VERY IMPORTANT)

👉 Rule: Correct order: Prefix(es) + Parent + Primary suffix + Secondary suffix

Example: CH3–CH(CH3)–CH2–OH

👉 3-Methylbutan-1-ol

🔹 Common Mistakes:

  • ❌ methylbutanol
  • ❌ butanol-3-methyl

✔ Always structured form.

⚡ FINAL MASTER EXAMPLE (Section 6 - Prefix & Suffix)

Compound: HO–CH2–CH(CH3)–CH2–COOH

Steps:

  1. Priority → COOH
  2. Chain → 4 carbons
  3. Substituents → OH, methyl
  4. Number from → COOH

👉 Final: 4-Hydroxy-3-methylbutanoic acid
